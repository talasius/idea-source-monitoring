# Дизайн для проведения этапа мониторинга источника
Список стимулов индивидуален для каждого участника

## Начало работы с PsychoPy

Для начала, установите [PyschoPy v.2026.1.0](https://www.psychopy.org/download.html)

- [Скачайте](https://github.com/talasius/idea-source-monitoring/releases/tag/ism_v1.0) архив с экспериментом со страницы последнего релиза
- Распакуйте ахрхив в любую папку
- Откройте `/idea_eval.psyexp` в среде PsychoPy Builder для работы с GUI, или же
- Откройте `/idea_eval.py` в среде PsychoPy Coder для работы с кодом

## Дополнительная информация

В папке `/stimuli` хранятся `.csv` файлы с персонализированными стимулами для каждого участника по их `ID`
При проведении эксперимента, они должны быть заменены на реальные списки участников

Папка `/data` будет создана автоматически после завершения эксперимента для сохранения данных

## Важно!

❌ Не запускайте онлайн версию эксперимента (JavaScript) - она не поддерживается

✅ Для корректной работы запускайте только `/idea_eval.psyexp` в среде PsychoPy Builder или `/idea_eval.py` в среде PsychoPy Coder
